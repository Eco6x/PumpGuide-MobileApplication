# 💪 PumpGuide

A fitness companion app built with Flutter. Browse exercises by muscle group, view detailed guides with animated GIF demonstrations, and generate randomized daily workouts.

## Features

- **Muscle Group Browser** — Filter exercises by Upper Body, Lower Body, Back, and Core
- **Exercise Details** — View full descriptions with typewriter animation, images, and GIF toggle
- **Random Workout Generator** — Get a random 5-exercise workout with one tap
- **Completion Tracking** — Mark exercises as complete during your session
- **Estimated Time** — See how long each exercise takes
- **Splash Screen** — Animated logo intro with fade-in text
- **Light/Dark Theme** — Automatically follows system theme

## Tech Stack

- **Framework:** Flutter (Dart)
- **Packages:** animated_text_kit, font_awesome_flutter
- **Platform:** Android

## Getting Started

**Prerequisites:** [Flutter SDK](https://flutter.dev/docs/get-started/install) and an Android emulator or physical device.

   ```bash
   git clone https://github.com/Eco6x/PumpGuide.git
   cd PumpGuide/pumpguide
   flutter pub get
   flutter run
   ```
